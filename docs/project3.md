# 候选人名单

添加候选人的方式有两种：

方法一：逐个添加候选人（姓名、性别和应聘岗位为必填项）

![PNG](image/p3-0.png)

方法二：导入候选人名单（下载模版，在模版的excel表格中填写候选人的信息，其中姓名为必填项）

![PNG](image/p3-1.png)
