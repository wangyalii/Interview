# 创建项目 #

**如何创建一个面试项目？**

- **一、创建项目**

- **二、在该项目里添加面试日程**

![PNG](image/p2-1.png)

- **三、在日程里添加面试官小组**

    注：添加面试官小组时需要选择该面试官小组使用的评分表和面试官，需要提前创建好评分表和面试官。
![PNG](image/p2-2.png)

- **四、添加候选人**
    
    在候选人名单添加候选人，可逐个添加也可批量导入。
    
    如需将候选人分组可在“候选组管理”进行分组。

- **五、添加面试安排**
    
    为面试官小组添加面试安排，该安排添加后面试官打分系统里会显示。
    
    设置面试时间和面试对象（面试对象可为多个候选人也可为候选组）

![PNG](image/p2-3.png)
